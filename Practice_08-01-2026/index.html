CSS Selectors:

1. Tag Selector

Class Selector:

Slects elements with a specific class , We use (.) to select class

Id selector:

Selects an element with unique id , we use(#) to select id

Universal Selector:

Selects all Html Elements on the page , we Use (*) to select

Group Selector:

Selects Multiple Elements at Once , and we seperate it by using comma(,)

Child Selector:

Slects Direct child Elements only

Adjacent sibling selector:

Selects The immediate Next Sibling and We use(+) to select

Genenral Sibling Selector :

Selects all Sibling elements after a specified element

We use (~) symbol to select all sibling Elemnts
